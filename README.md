## iStore POS-App With AI

- **Features:**

- **ร้านค้า**
  - การเพิ่มร้านค้าใหม่ ภายในองค์กร
- **การยืนยันตัวตร:**
  - การเข้าสู่ระบบ
  - การออกจากระบบ
  - การแก้ไขข้อมูลส่วนตัว
- **แดชบอร์ด:**
  - สินค้าใกล้จะหมด
  - chart ยอดขายรายปี
  - อันดับสินค้าขายดี
- **การขาย:**
  - ตะกร้าสินค้า
  - แนะนำโปรโมชั่นใกล้เคียง
  - การเพิ่มสินค้าไปยังตะกร้าโดยการสแกน Barcode
  - การเพิ่มสินค้าโดยการค้นหา
  - คำนวณเงินทอน
  - เลือกประเภทการโอน
  - ขายประเภทต่างๆ เช่น การขายปกติ, ขาย Pre-order, ฝากขาย
- **Pre-order:** การขายสินค้าโดยไม่จำเป็นต้องมีสินค้าในสต๊อกและมาจัดการในภายหลังเมื่อมีสินค้า
- **ฝากขาย:** การขายสินค้าแบบ หักสต๊อกชั่วคราว เพื่อนำไปขายข้างนอกร้านและเมื่อขายไม่หมด สามารถมาจัดการคืนสต๊อกได้
- **สมาชิก Member:**
  - การสะสมแต้ม
- **สินค้า:**
  - กูคืนสินค้า
  - สร้าง Barcode สินค้าสำหรับสินค้า DIY หรือไม่มี Barcode
  - Generate Barcode to png
  - ควบคุมการ พรีออเดอร์ และการจัดการสต๊อก
- **การจัดการสต๊อก** การสร้างรายการสต๊อกสำหรับการเพิ่มสต๊อกสินค้าหลากหลายสินค้า พร้อมกัน
- **ประวัติการขาย:**
  - ตรวจสอบผู้ขาย
  - Filter การขาย วันเริ่มต้น, วันสิ้นสุด, วิธีชำระเงิน, ผู้คิดเงิน, ยอดขั้นต่ำ, ยอดสูงสุด และหมายเหตุ
- **หมวดหมู่สินค้า** จัดการหมวดหมู่สินค้า
- **โปรโมชั่น** การจัดการโปรโมชั่น (ซื้อ X รับ Y)
- **ประชาสัมพันธ์:** การประชาสัมพันธ์โปรโมชั่นสินค้าไปยังแอพพลิเคชั่นต่างๆ
  - Ai Generate รูปภาพโปรโมชั่น
  - การตั้งเวลาประชาสัมพันธ์
- **แอพพลิเคชั่น:** จัดการแอพพลิเคชั่นที่เกี่ยวข้อง
  - Line Chatbot
  - Line ประชาสัมพันธ์
- **พนักงาน** จัดการพนักงาน
- **ตำแหน่ง** จัดการสิทธิ์และตำแหน่งของพนักงาน
- **บัญชี:** จัดการบัญชีผู้ใช้
  - เปลี่ยนข้อมูลผู้ใช้
  - เปลี่ยนรหัสผ่าน
- **ChatBot** ตอบคำถามเกี่ยวกับร้านค้า สินค้า โปรโมชั่น
